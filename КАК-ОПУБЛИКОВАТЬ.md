# Хозтовары Бати — как посмотреть и выложить бесплатно

## Посмотреть на компьютере

Откройте файл `index.html` в браузере (двойной клик или перетащить в Chrome).

## Добавить фото

1. Скопируйте фото в папку `assets/images/`.
2. Назовите главные: `hero-main.jpg`, `hero-sub.jpg` (или поменяйте пути в `js/main.js`).
3. Для галереи откройте `js/main.js` и в массив `GALLERY_IMAGES` добавьте пути, например:

```js
const GALLERY_IMAGES = [
  "assets/images/porolon-1.jpg",
  "assets/images/kleenka-1.jpg",
];
```

## Бесплатный хостинг (GitHub Pages)

1. Создайте репозиторий на GitHub (например `hoztovary-bati`).
2. Загрузите содержимое папки `Хозтовары-Бати` в корень репозитория.
3. Settings → Pages → Source: **main** branch, folder **/ (root)**.
4. Через 1–2 минуты сайт будет по адресу: `https://ВАШ_ЛОГИН.github.io/hoztovary-bati/`

Бесплатно, без домена. Свой домен — опционально и обычно платно.
